B 方向里，你更想要哪种克制感？

都是“桌面工具感”，但侧重点不同：是更轻、更稳，还是更密、更专业。

B1 · Airy / 轻、干净、舒展
Library · Albums · Artists · Settings
留白更多
边框更轻
更安静、更通透

适合长时间看列表和资料页

轻与舒展

更强调留白、通透和温和边框。看起来更“轻”，但不追求强烈个性。

B2 · Studio / 稍密、专业、稳定
Songs · Queue · Search · Settings
密度略高
信息更稳
更像专业桌面软件

层级靠边框、密度和字重，不靠强颜色

专业与稳定

更强调秩序、密度和稳态层级，适合播放器这种长期驻留的桌面应用。

B3 · Soft Contrast / 轻工具感 + 一点精致高光
Library · Now Playing · Settings
基础克制
局部高光
大体克制，但保留少量品牌感

比 B1/B2 更“精致”，也更容易跑偏

克制里保留一点高光

总体仍是桌面工具感，但允许顶部、播放器、重点卡片有少量精致亮点。

点一个最接近的方向；如果介于两者之间，直接在终端说组合也行，比如“B2 为主，带一点 B3”。
