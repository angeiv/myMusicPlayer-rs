# Rust 跨平台桌面音乐播放器 - 详细开发任务清单

## 项目概览
基于 Tauri + Rust + Svelte 的跨平台桌面音乐播放器，支持多种音频格式，提供现代化的用户体验。

## 技术栈
- **后端**: Rust + Tauri
- **前端**: Svelte + TypeScript + Vite
- **音频处理**: Symphonia + Rodio
- **数据库**: SQLite (通过 `rusqlite`)
- **构建工具**: Cargo + npm + just
