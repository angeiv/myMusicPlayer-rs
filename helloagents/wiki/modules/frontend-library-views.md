# frontend-library-views

## 目的
负责歌曲、专辑、艺人和播放列表等列表与详情页展示。

## 模块概述
- 职责: 查询曲库数据、展示详情、发起播放动作
- 状态: ✅稳定
- 最后更新: 2026-03-20

## 规范

### 需求: 曲库浏览
**模块:** frontend-library-views
各视图需要使用统一的 DTO 呈现曲库数据，并允许从列表项触发播放。

#### 场景: 歌曲列表发起播放
- 前置条件: 曲库数据加载完成
- 预期结果: 用户可从歌曲或详情页触发播放，底栏状态随后刷新
